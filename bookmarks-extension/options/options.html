How sync works

  1. Reads this browser’s bookmark tree (URLs + folder paths).
  2. Talks to POST /api/bookmarks/sync (or downloads the list), depending on strategy.
  3. Applies the result back to the browser when needed.

Create a user in the server admin UI, copy that user’s API key, and paste it above. Brave and Chrome both load this as an unpacked Manifest V3 extension.
