# Privacy Policy — Bookmarks Sync browser extension

See the canonical policy (for Chrome Web Store and users):

- **Markdown:** [../docs/PRIVACY.md](../docs/PRIVACY.md)
- **HTML (GitHub Pages / raw):** [../docs/privacy.html](../docs/privacy.html)

**Short answer:** Yes, the extension handles user data (bookmarks + settings) so it can sync with *your* server. It does not send data to the developer by default. A public privacy policy URL is required for the Chrome Web Store.
